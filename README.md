# Closed-List: 個人化封閉式清單應用程式

這是一個基於生產力專家馬克·福斯特 (Mark Forster) 在其著作《明天再做》(Do It Tomorrow) 中提出的「封閉式清單」(Closed List) 方法所打造的個人化任務管理網頁應用程式。

它的核心目的是為了解決傳統待辦清單「永遠做不完」的問題，透過為每日工作設定明確的界線，來幫助使用者提高專注力、減少焦慮並獲得每日的成就感。

## ✨ 核心功能

*   **手動建立清單**：每日的「封閉式清單」將由您親手建立，確保您對當天的工作有完全的掌控。
*   **延後處理新任務**：當天出現的新任務會被放入一個獨立的「明天待辦」區塊（以淡化方式顯示），不會干擾今日工作，並為明日的規劃做好準備。
*   **緊急任務插入**：提供一個特殊的「緊急插入」功能，允許極少數的緊急任務被加入今天的清單，並會有特殊標記。
*   **未完成任務自動滾動**：今天未完成的任務，將在第二天自動移動到新清單的最頂部，確保它們被優先處理。
*   **歷史回顧**：您可以方便地查閱過去每天的封閉式清單及任務完成狀況，便於您進行回顧與改善。
*   **跨平台使用**：作為一個網頁應用程式，您可以在任何有瀏覽器的裝置上使用，包括電腦和手機。
*   **離線優先**：所有資料儲存在您的瀏覽器本地，無需網路連線也可正常使用。

## 🚀 開始使用 (Getting Started)

依照以下步驟在您的本機環境啟動這個專案：

1.  **複製專案庫**
    ```bash
    git clone https://github.com/your-username/closed-list.git
    cd closed-list
    ```

2.  **安裝依賴**
    ```bash
    pnpm install
    ```

3.  **啟動開發伺服器**
    ```bash
    pnpm dev
    ```

4.  在瀏覽器中開啟 `http://localhost:4321` 即可看到應用程式。

## 📖 「封閉式清單」核心理念

本應用程式遵循以下核心原則：

1.  **創建並「封閉」**：為今天建立一個任務清單，然後「封閉」它，不再增加。
2.  **延後新任務**：所有新出現的任務都被延後到明天處理。
3.  **專注完成**：專注於完成今天清單上的所有任務，享受「清空」的成就感。
